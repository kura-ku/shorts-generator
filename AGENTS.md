# AGENTS

## Project Goal

Build a fully automated YouTube Shorts generator.

## Rules

Never duplicate code.

Prefer extending an existing Engine.

Maintain backward compatibility.

Always update ROADMAP.md after adding a feature.

Do not hardcode episode names.

Keep functions small.

Keep UI separated from business logic.

Subtitle rendering belongs to Subtitle Engine.

Scene switching belongs to Scene Engine.

Rendering belongs to Render Engine.
